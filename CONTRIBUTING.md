# Contributing

Thanks for your interest in contributing to `open-campus-unreal-90min`.

## Scope

This repository stores the standalone lesson materials for a 90-minute Unreal
Engine open campus experience class. Keep it content-first and repository-local.

## Setup

1. Install Node.js 22 or later.
2. Run `npm install`.
3. Run `compose-agentsmd`.
4. Run `git config core.hooksPath .githooks`.

## Verification

Run the full verification suite before each commit:

- `npm run verify`
- `compose-agentsmd`
