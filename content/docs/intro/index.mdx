Unreal Engine 5.x と Blueprint を使った、オープンキャンパス向け 90 分体験授業の教材です。
短時間で「動くゲームを完成させる」体験を作ることを目的にしています。

## この教材でできること

- Unreal Engine 5.x の Third Person テンプレートを使って授業を実施できる
- 受講者にそのまま渡せる手順書を使って 90 分授業を回せる
- 教員向けの進行台本と、教室準備・障害時フォールバックを参照できる

## 教材構成

- [授業概要](/docs/lesson-overview)
- [受講者ガイド](/docs/student-guide)
- [教員ガイド](/docs/teacher-guide)
- [セットアップ・トラブル対応](/docs/setup-and-troubleshooting)

## 想定する授業の流れ

1. 教員が [教員ガイド](/docs/teacher-guide) を読んで進行を決める
2. 教室準備と当日運用を [セットアップ・トラブル対応](/docs/setup-and-troubleshooting) で確認する
3. 授業中は受講者に [受講者ガイド](/docs/student-guide) を見せながら進める

## 公開と運用

この repo は `metyatech/course-docs-site` が読む content repository です。公開環境では
Vercel 上の共有 runtime にこの repo の内容を同期して表示します。

教材の正本は GitHub repository の履歴です。授業現場ではサイトを投影しつつ、必要に応じて印刷物や補助資料を併用します。
