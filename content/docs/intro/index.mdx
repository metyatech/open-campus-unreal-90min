この教材は、**オープンキャンパスでゲーム分野に興味を持った初学者**に向けた、**90分の体験授業**のための資料です。

**この授業のゴールは、難しいことを全部理解することではなく、自分で作った仕組みでゲームが変わる体験を持ち帰ること**です。

## 最初にわかること

- 対象: 主に高校既卒以上の初学者
- 時間: 90分
- 使うもの: Unreal Engine 5.x / Third Person テンプレート / Blueprint
- 完成物: アイテムを集めてゴールするとクリアになるミニゲーム

## この授業で大事にすること

### 1. まず遊べる状態を作る

**最初から細かい作り込みはしません。** 先に動く土台を作ることで、「作れた」という実感を早く得られます。

### 2. タイピングより仕組みの理解を優先する

C++ ではなく Blueprint を使うのは、**処理の流れを目で見ながら学べる**からです。

### 3. 全員が完走できる設計にする

この授業はコンテストではありません。**完成度より、90分で最後までたどり着けること**を優先します。

## この授業で扱わないこと

- C++ の文法学習
- 3D モデルやアニメーションの作成
- 長時間の作り込み
- 本格的なUI設計やセーブ機能

**90分で必要なのは、「全部学ぶこと」ではなく、「ゲームはこう作るのか」と実感することです。**
